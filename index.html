About

Sebagai IT Support dan Web Programmer, saya memiliki pengalaman luas dalam mendukung dan mengembangkan solusi teknologi informasi untuk bisnis dan organisasi. Saya memiliki keahlian dalam memecahkan masalah, mendiagnosis, dan memperbaiki masalah perangkat keras dan perangkat lunak, serta pengalaman mengelola infrastruktur jaringan. Selain itu, saya juga berpengalaman dalam mengembangkan aplikasi web, baik front-end maupun back-end, serta memastikan keamanan dan skalabilitas situs web. Saya dapat bekerja dengan berbagai bahasa pemrograman seperti HTML, CSS, JavaScript, PHP, dan sudah terbiasa menggunakan framework Laravel.
